# OWL Vocabulary

This directory contains the OWL ontology definitions for KTDDE trade documents.

## Files

- `ktdde.owl` - Main KTDDE ontology (to be added)
- `ktdde.ttl` - Turtle serialization (to be added)

## Usage

The ontology defines:
- Classes for trade document types
- Properties for document fields
- Relationships between entities
- Data type constraints
- Business rules and validation logic
